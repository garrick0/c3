# Projection Context

Transforms property graphs into analytical views for different use cases.

## Entities

- **Projection** - Base projection concept
- **ModuleProjection** - Module-level view
- **LayerProjection** - Architectural layer view
- **DependencyMatrix** - Dependency analysis
- **ComponentGraph** - Component relationship view
- **TreeProjection** - Hierarchical tree view

## Services

- **ProjectionEngine** - Core projection logic
- **GraphTransformer** - Graph transformation algorithms
- **NodeAggregator** - Node merging and grouping
- **MetricsCalculator** - Calculate view-specific metrics
- **LayoutEngine** - Layout algorithms for visualization

## Ports

- **ProjectionStrategy** - Strategy for different projections
- **Renderer** - Rendering interface
- **Exporter** - Export to different formats
- **ViewRepository** - Storage for computed views
